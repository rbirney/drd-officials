# Subquery with IN

This query will return multiple results:

~~~sql
SELECT serialNumber
FROM Equipment 
WHERE equipDescription LIKE '%Mat%';
~~~

If we want to use the results of this query to find out, say, which classes use these pieces of equipment we can use a subquery with the IN keyword.

~~~sql
SELECT classequipment.classId, fitnessclass.classDescription, classequipment.serialNumber
FROM classequipment 
JOIN fitnessclass
ON classequipment.classId = fitnessclass.classId
WHERE serialNumber IN (
	SELECT serialNumber
	FROM Equipment 
	WHERE equipDescription LIKE '%Mat%'
);
~~~

The outer query selects the classId, classDescription, and serialNumber from the classequipment and fitness class tables (using a JOIN). The WHERE clause uses the results of the subquery and shows results where there is a matching serialNumber. The subquery returns the results "EQ1002" and "EQ1010" so this is equivalant to saying:

~~~sql
WHERE serialNumber IN ('EQ1002', 'EQ1010')
~~~
