# IN

The *IN* operator allows us to query records based on matching one of a number of values. We can use the *NOT* operator to negate those results. So if we are only interested in Equipment from NordicTrack, Liforme and Concept2, 
instead of writing the query as follows:

~~~sql
SELECT * 
FROM Equipment 
WHERE make = 'NordicTrack' OR make = 'Liforme' OR make ='Concept2';
~~~

We can write it as follows:

~~~sql
SELECT * 
FROM Equipment 
WHERE make IN ('NordicTrack', 'Liforme', 'Concept2');
~~~

## NOT IN

To select from outside the above range, use:

~~~sql
SELECT * 
FROM Equipment 
WHERE make NOT IN ('NordicTrack', 'Liforme', 'Concept2');
~~~
