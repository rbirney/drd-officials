# AND/OR

When one search criteria is not enough we use the *AND*, *OR* operators to use more than one search condition.
The following commands give very different results - can you work out why? 

~~~sql
SELECT * 
FROM GymMember 
WHERE county = 'Waterford' AND lastName = 'McGrath';
~~~

~~~sql
SELECT * 
FROM GymMember 
WHERE county = 'Waterford' OR lastName = 'McGrath';
~~~

~~~sql
SELECT * 
FROM GymMember 
WHERE county = 'Waterford' OR county = 'Wexford';
~~~
