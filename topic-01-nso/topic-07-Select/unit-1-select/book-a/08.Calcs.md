# Calculations in our database tables

## DATEDIFF()

The DATEDIFF() function is used to return the distance (number of days) between two dates.

The following example returns the number of days until 2030 (January 1, 2030):

```sql
SELECT DATEDIFF('2030-01-01', CURDATE()) AS "Number of days until 2030";
```


## TIMESTAMPDIFF()

This function in MySQL is used to return a value after subtracting a DateTime expression from another. It returns the result in the specified unit (Years, Quarters, Months, Days, Hours, Minutes, Seconds, etc..)

For example, to calculate a person's age use:

```sql
SELECT  TIMESTAMPDIFF(YEAR, dateofbirth, CURDATE()) AS age;
```

**Note:** You could replace *dateofbirth* with an actual date of birth.

In the following example, we return the gym member (by name), and their age.

```sql
SELECT CONCAT(firstname, " ",  lastname) AS Member, TIMESTAMPDIFF(YEAR, dateOfBirth, CURDATE()) AS Age 
FROM gymmember
ORDER BY lastname, firstname;
```

## DATE_ADD()

This function in MySQL is used to add a time/date interval to a date and then return the date. 

In the following example, we return the gym member (by name), and their membership renewal date based on their stored current renewal date. 

```sql
SELECT CONCAT(firstname, ' ',  lastname) AS Member, DATE_ADD(renewalDate, INTERVAL 1 YEAR) AS "Renewal Date" 
FROM gymmember
ORDER BY lastname, firstname;
```

In the following example, we return member (by name), their renewal date, their next renewal date, and a boolean flag to check whether their renewal is due.

```sql
SELECT CONCAT(firstname, ' ',  lastname) AS Member,  renewalDate, 
	   DATE_ADD(renewalDate, INTERVAL 1 YEAR) AS nextPaymentDue,
       CURDATE() >= DATE_ADD(renewalDate, INTERVAL 1 YEAR) AS isDue
FROM gymMember
ORDER BY lastname, firstname;
```

## Using Operators

What if we want to use SQL to calculate a value that is not stored anywhere in the database? We can just use statements as follows:

~~~sql
SELECT 12 + 14; // Addition
SELECT 12 - 14; // Subtraction
SELECT 12 * 14; // Multiplication
SELECT 12 / 14; // Division
SELECT 12 % 14; // Modulus
SELECT 14 % 12; // Modulus
~~~
