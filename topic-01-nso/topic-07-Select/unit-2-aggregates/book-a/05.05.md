# HAVING

In order to restrict rows returned with aggregate functions, you must use a *HAVING* clause. Essentially it is a WHERE clause for aggregate functions.

## Example

The following example returns the number of members grouped by (per) county for counties that have 5 or more students.

~~~sql
SELECT county, COUNT(memberId) AS "Number of Members"
FROM Gymmember 
GROUP BY county
HAVING COUNT(memberId) > 5;
~~~
