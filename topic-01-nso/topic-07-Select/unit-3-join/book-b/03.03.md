# Multi-table Example Two

We can expand the above example to return the Trainer of the Class also.
We want to retrieve the list of all classes (class description) that a member takes (firstname and lastname) and
the trainer (firstname and lastname) of the class.

There are three things to do here:

1. Identify which tables have the data you are looking for, member name is in the *gymmember* table, class description is in the *fitnessClass* table, and trainer name is in the *Trainer* table. The tables required are: *gymmember*, *fitnessClass*, *memberClass*, and Trainer. Since there are 4 tables, we need 3 joins. 

2. Identify which columns are the primary and foreign keys in these tables. We use these in the *ON* clause.
    - There are no foreign keys in *gymmember* or *fitnessClass*, so we cannot join these tables together but there are 2 foreign keys in *memberClass*, which links members to classes. 
    - To join the gymmember and memberClass tables we use *memberId*, which is the Primary key in gymmember and a foreign key in memberClass. 
    - To join the fitnessClass and memberClass tables we use *classId*, which is the Primary key in fitnessClass and a foreign key in memberClass.
    - To join the fitnessClass and Trainer tables we use *PPS*, which is the Primary key in Trainer and a foreign key in fitnessClass.		

3. Identify which columns we want the query to output; in this case it is *firstName*, *lastName* from the *gymmember* table, *classDescription* from the *fitnessClass* table, and *firstName*, *lastName* from the *Trainer* table. We will use these in our *SELECT*.

~~~sql
SELECT CONCAT(g.firstname, " ", g.lastName) AS Member, classDescription Class, 
       CONCAT(t.firstname, " ", t.lastName) AS Trainer
FROM gymmember g JOIN memberClass 
ON g.memberId = memberClass.memberId
JOIN fitnessClass 
ON fitnessClass.classId = memberClass.classId
JOIN trainer t
ON t.PPS = fitnessClass.PPS
ORDER BY g.lastname, g.firstname;
~~~

Could be written as:

~~~sql
SELECT CONCAT(g.firstname, " ", g.lastName) AS Member, classDescription Class, 
       CONCAT(t.firstname, " ", t.lastName) AS Trainer
FROM gymmember g JOIN memberClass USING (memberId)
JOIN fitnessClass USING (classId)
JOIN trainer t USING (PPS)
ORDER BY g.lastname, g.firstname;
~~~
