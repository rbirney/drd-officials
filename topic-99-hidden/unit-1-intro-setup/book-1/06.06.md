# 6. Commit and push changes

Now that we have created our first React app, let's save the changes to our Git repository. These steps are the same as what we did at the end of Step 4, and you will repeat these steps each time you make changes to your app.

## Committing changes

**IMPORTANT**: Run these commands in the "react-basics-labs" folder (you may need to use `cd ..\` to go back up a level to that folder).

~~~bash
git add -A
git commit -m "created new react app"
git push origin main
~~~

Verify that the changes appear on GitHub.com.
